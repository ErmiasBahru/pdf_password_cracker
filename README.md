# pdf password cracker

### A simple python program that crack a password protected pdf file

[live demo](https://vimeo.com/722973220)

### Required:

1. pikedpdf
2. tqdm

### Install the required dependencies:

`$ pip3 install pikepdf tqdm`

### Usage:

`$ python pdf_cracker.py`
